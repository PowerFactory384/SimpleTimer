# Simple Timer
- Just a simple timer that can count time and some other features.

![test](./img/test.png)

## Features
- Time add button
- Edit Time UI
- Stop / Start button
- Reset button
- Stopwatch button
- Light / Dark Mode

## Used Library
- [Scrrible](https://www.jujuadams.com/Scribble/#/latest/)
- delta
